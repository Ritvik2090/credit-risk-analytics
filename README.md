# Credit Risk Analytics

## Overview
This project analyzes borrower data and builds machine learning models to predict loan default risk.

## Objectives
- Understand factors influencing loan defaults
- Perform exploratory data analysis
- Engineer risk-related features
- Build and evaluate classification models
- Generate actionable business insights

## Tech Stack
- SQL
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
